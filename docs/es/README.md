# NBTMap Explorer

**NBTMap Explorer** es una herramienta diseñada para explorar, previsualizar y exportar objetos de mapa del juego desde mundos guardados de Minecraft. Permite a los usuarios visualizar metadatos almacenados en formato NBT y extraer imágenes de mapas en varios formatos.

## Agradecimientos

Este proyecto fue inspirado por [Shokuno](https://github.com/Shokuno), quien originalmente creó una herramienta similar en Java. Esta versión expande esa idea utilizando C# y WPF, con el objetivo de ofrecer una mejor experiencia de usuario y funciones adicionales.

- Agradecimientos especiales a [Shokuno](https://github.com/Shokuno) por el concepto original y su implementación en Java.
